Wife's Recipe Book
This is an app I build for my wife. It is a recipe book, as its name suggests and it is still in the development phase. I build it using Angular 19 and Strapi for data storage and security, using Google as the only authentication (maybe for now). You will get 4 daily random selected recommended recipes, you can add your own recipes, edit and delete only yours, but you can see everyone's recipes. I am currently working on adding more functionalities and redesigning the entire app.

Reconcilify
Internal tool for a Swiss-based travel retailer which operates duty-free and duty-paid shops and convenience stores. The app contains 3 modules that manage the vendors and customers data(Finance), the categories and groups of food and beverages for a global chain of restaurants(F&B) and the former employees for audit and account deactivation.
